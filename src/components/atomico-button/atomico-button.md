# atomico-button

Just another example ... what is special about it? the use of `is="atomico-button"`, fully supported by Atomico.

## Installation

```html
<script src="http://unpkg.com/@atomicojs/components/atomico-brand.js"></script>
```

## Use

<doc-show-html show>
    <a is="atomico-button" background="black" color="white">my button</a>   
</doc-show-html>

## Properties

<doc-props selector="a[is=atomico-button]"></doc-props>
